# Lighthouse Technology Sigma Detection Repository

## Purpose

This repository contains vendor-neutral Sigma detection rules developed under
Phase G - Detection Engineering & Coverage Measurement.

## Standards

All Sigma rules must include:

- Detection ID
- Rule ID
- MITRE ATT&CK mapping
- Telemetry requirements
- False-positive guidance
- Validation requirements
- Evidence references
- Lifecycle metadata
